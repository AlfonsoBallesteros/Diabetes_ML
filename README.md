# Diabetes_ML
Bulding model with data set pima-indians-diabetes

## Prerequisite
    1. pandas
    2. numpy
    3. Matplotlib
    4. scikit-learn
        4.1. LogisticRegression
        4.2. KNeighborsClassifier
        4.3. GaussianNB
        4.4. SVC
        4.5. LinearSVC
        4.6. RandomForestClassifier
        4.7. DecisionTreeRegressor
    5. Backend: Flask
    6. Frontend: Vanilla Javascript

## Install

```sh
pip install
```

## Usage

```sh
python app.py
```
